# 📊 Data Science Repository

## 📌 About this repository

This repository brings together practical Python applications focused on solving common and well-known Data Science problems. The goal is to demonstrate, in a clear and applied way, techniques frequently used in real-world scenarios.

Among the topics covered, you will find:

- Data Science problem-solving approaches  
- Random sampling techniques for data analysis  
- Data cleaning and preprocessing strategies  
- Validation and standardization of monetary values using regex  
- Data Mining concepts and applications  
- Machine Learning fundamentals and examples  
- Practical implementations using Python  
- Exercises and datasets inspired by Kaggle  

## 🎯 Purpose

The main objective of this repository is to serve as a hands-on reference for:
- Improving data manipulation and validation skills  
- Applying statistical and computational methods  
- Showcasing practical solutions to common data challenges  

## 🛠️ Technologies

- Python  
- Pandas  
- Regular Expressions (Regex)  
- Machine Learning libraries
